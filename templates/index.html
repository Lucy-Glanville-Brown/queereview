Code for Proud people!

Welcome to QueeReview

QueeReview is a website for the LGBTQ+ tech community where they can find help with queries on specific coding issues, request peer reviews of projects, and ask the community about LGBTQ+ issues related to the tech workplace or any general life issues they may have. QueeReview is a place offering tech, work and life support.

How Reviews Work

We have 3 types of posts held within our Review Stream

This is a safe place to ask questions, submit full peer reviews and get help with code snippets and functions

General

General Posts are for questions, events, advise!!! Maybe you want to ask a question about an LGBTQ issue at work, or you want to share your weekly win with the community! What ever it is go ahead!

Code

Code Snippet Requests are for other users to take a quick look at short pieces of code to help you trouble shoot your issues. We advise using Code Pen You will need to set up an account and then once you've created a codepen you can share the link in the bottom right hand corner of the code pen. If you have another link to share a snippet please feel free to use that instead.

Review

Review Post are for requests for the community to review your entire project, if there is something specific you'd like adivse on then please let the community know in the description. Include links to your repo and your deployed site so it can be tested properly.
